# 飞书端配置指南（手动操作）

## 第一步：创建飞书开放平台应用

1. 打开 **[飞书开放平台](https://open.feishu.cn)** → 点击"创建应用"
2. 应用名称：`私募拜访数据同步`（随意）
3. 创建后进入应用详情页

## 第二步：开通权限

1. 左侧导航 → **权限管理** → 搜索并添加以下权限：

   | 权限 | 用途 |
   |------|------|
   | `bitable:app` | 读写多维表格 |
   | `bitable:app.table` | 读写多维表格数据 |
   | `contact:user.base` | 读取用户信息（获取提交人身份） |

2. 点击"批量开通"

## 第三步：获取应用凭证

1. 左侧导航 → **凭证与基础信息**
2. 记下 **App ID** 和 **App Secret**（后面 n8n 配置要用）

## 第四步：发布应用（测试环境）

1. 左侧导航 → **版本管理与发布** → 创建版本
2. 填写版本号（如 1.0.0），发布说明随便填
3. 点击"发布"（如果公司有审批流程，需要审批通过）
4. 应用需要"企业自建应用"类型，管理员审批即可

## 第五步：创建多维表格

1. 打开飞书 → 多维表格 → 新建空白多维表格
2. 重命名为 `私募拜访数据采集`
3. 按照 `bitable_schema.md` 中的字段结构创建所有字段

   快速创建指引：
   - 默认已有"字段1"文本 → 改名 `机构名称`
   - 新增字段 → `登记编号`（文本）
   - 新增字段 → `拜访日期`（日期）
   - ...以此类推
   - 最后加 `同步状态`（单选），选项：未同步、已同步、失败

4. 设置 `同步状态` 字段默认值为 `未同步`

## 第六步：创建表单

1. 在多维表格右上角点击 **表单** → **新建表单**
2. 表单标题：`私募拜访记录录入`
3. 选择需要显示在表单中的字段（删除自动字段如 同步状态、同步日志、提交人）
4. 设置必填字段：机构名称、拜访日期、拜访人、拜访状态
5. 点击"发布"，复制表单链接发给同事们

## 第七步：获取多维表格的 app_token 和 table_id

打开该多维表格，URL 格式为：
```
https://xxx.feishu.cn/base/{app_token}?table={table_id}
```

**app_token**: `base/` 后面的字符串
**table_id**: `table=` 后面的字符串

## 第八步：配置到本地

将获取到的信息填入配置文件：

```bash
# 编辑 feishu_sync/config.env
FEISHU_APP_ID=cli_xxxxx           # 从飞书开放平台获取
FEISHU_APP_SECRET=xxxxx           # 从飞书开放平台获取
FEISHU_APP_TOKEN=xxxxx            # 多维表格 app_token
FEISHU_TABLE_ID=xxxxx             # 多维表格 table_id
```

---

## 验证方式

填表后，确认多维表格中能看到新记录，且 `同步状态` 列显示"未同步"。
